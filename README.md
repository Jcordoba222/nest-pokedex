<h1>Ejecutar desarrollo<h1>

<h3>1. Clonar repositorio<h3>
<h3>2. Ejecutar<h3>

```
npm install
```
<h3>3. Tener Nest CLI instalado<h3>

```
npm i -g @nestjs/cli
```
<h3>4. Levantar la base de datos<h3>

```
docker-compose up -d
```
